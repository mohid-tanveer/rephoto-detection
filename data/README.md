# Data

## Overview

Consists of 2 folders of images: authentic and re-photo. re-photos are separated by type of screen (LCD and OLED) as well as by the source of the re-photo (AI and authentic).

## Structure

```bash
data/
├── authentic/
├── re-photo/
│   ├── ai-LCD/
│   ├── ai-OLED/
│   ├── authentic-LCD/
│   └── authentic-OLED/
```
